InfernalSynth

An open source semi-modular VST3 synthesizer and effect plugin.

Requirements

64-bit Windows.
Tested in Renoise, REAPER, Bitwig Studio and FL Studio.
Distributed as VST3, no other plugin formats supported.

Download

InfernalSynth 1.1: release/debug and demos.
SevenSynth 1.0: release/debug and demos.
When in doubt, you want the release versions.

Install

Extract the zip archive and copy the InfernalSynth folder to C:\Program Files\Common Files\VST3.
Overwrite anything that may already be there. Versioned builds can coexist with other versioned builds,
but there can only be one generic build installed. Unless you plan to run SevenSynth and InfernalSynth
(versioned) side-by-side, also remove SevenSynth from your VST3 folder prior to installing InfernalSynth.
