InfernalSynth

An open source semi-modular VST3 synthesizer and effect plugin.

Requirements

64-bit Windows 7+ and a CPU with AVX support.
Tested in Renoise, REAPER, Bitwig Studio and FL Studio.
Distributed as VST3, no other plugin formats supported.

Install

Delete any previous installations first. Extract the zip archive and copy the InfernalSynth folder(s) to C:\Program Files\Common Files\VST3. For windows 7 and 8, you'll also need Microsoft's VC++ redistributable package, see the X64 download over here: https://learn.microsoft.com/en-US/cpp/windows/latest-supported-vc-redist?view=msvc-170.

Note: you have to keep the folder structure intact, including binaries, presets and themes. VST3's are bundles, not files.
