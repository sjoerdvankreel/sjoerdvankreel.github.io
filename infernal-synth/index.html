InfernalSynth

An open source semi-modular VST3 synthesizer and effect plugin.

Requirements

64-bit Windows 7+ and a CPU with AVX support.
Tested in Renoise, REAPER, Bitwig Studio and FL Studio.
Distributed as VST3, no other plugin formats supported.

Install

If you were using a previous "Generic" version, delete that first. Extract the zip archive and copy the InfernalSynth folder to C:\Program Files\Common Files\VST3. Overwrite anything that may already be there. Versioned builds can coexist with other versioned builds, but there can only be one generic build installed. For windows 7 and 8, you'll also need Microsoft's VC++ redistributable package, see the X64 download over here: https://learn.microsoft.com/en-US/cpp/windows/latest-supported-vc-redist?view=msvc-170.
