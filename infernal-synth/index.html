InfernalSynth

An open source semi-modular VST3 synthesizer and effect plugin.

Demos

Requirements

64-bit Windows.
Tested in Renoise, REAPER, Bitwig Studio and FL Studio.
Distributed as VST3, no other plugin formats supported.

Download and install

InfernalSynth 1.1

InfernalSynth 1.1 or InfernalSynth 1.1 debug.

SevenSynth 1.0 (renamed)

SevenSynth 1.0 or SevenSynth 1.0 debug.
When in doubt, you don't want the debug version.
Extract the zip archive and copy the InfernalSynth folder to C:\Program Files\Common Files\VST3.
Please note, versioned builds can coexist with other versioned builds, but there can only be one generic build installed.
