InfernalSynth

An open source semi-modular VST3 synthesizer and effect plugin.

Requirements

64-bit Windows.
Tested in Renoise, REAPER, Bitwig Studio and FL Studio.
Distributed as VST3, no other plugin formats supported.

About

Read the manual: https://github.com/sjoerdvankreel/infernal-synth/blob/main/MANUAL.md.
See the github page for details and source code: https://github.com/sjoerdvankreel/infernal-synth.
Play around with the demo projects: https://github.com/sjoerdvankreel/infernal-synth/tree/main/demo.

Download and install

InfernalSynth 1.1: Release/Debug.
SevenSynth 1.0: Release/Debug.

When in doubt, you don't want the debug versions.
Extract the zip archive and copy the InfernalSynth folder to C:\Program Files\Common Files\VST3.
Please note, versioned builds can coexist with other versioned builds, but there can only be one generic build installed.
