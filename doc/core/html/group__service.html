Opens an aggregate audio stream.

Returns
0 on success, a nonzero error code otherwise.
Parameters
sthe audio service.
devicespointer to an array of [count] devices.
channelspointer to an array of [count] channel specifications.
bufferSizespointer to an array of [count] buffer sizes in milliseconds.
countnumber of elements in the devices, channels, and bufferSizes arrays.
mixthe mix format, to be used by all streams in the aggregated stream.
interleavedopen the stream for interleaved access (true) or non-interleaved access (false).
masterthe master device, will be used for synchronization.
streamCallbackthe application-defined stream callback.
xRunCallbackthe application-defined xrun callback (may be NULL).
useruser data passed to the stream callback (may be NULL).
streamon success, receives a pointer to the opened audio stream. On error, *stream will be NULL.

Opens an aggregated audio stream on [count] devices. Each underlying stream is opened by calling XtDeviceOpenStream, passing in the Nth device, Nth buffer size and the combination of the specified mix format and the Nth channel specification as the audio format.

Aggregate streams use an intermediate buffer to keep underlying streams in sync. The total latency overhead of the aggregate stream is bounded by the largest buffer size of any of the underlying streams times a constant factor.

Use XtDeviceGetBuffer to query supported buffer sizes.

The resulting stream must be closed using XtStreamDestroy.

When the requested interleaving mode is not supported by any device it will be emulated.

This function may only be called from the main thread.

Open the default audio device.

Returns
0 on success, a nonzero error code otherwise.
Parameters
sthe audio service.
outputtrue to open the default output device, false to open the default input device.
deviceon success, receives a pointer to the opened device. On error or when no default device is present, *device is set to NULL.

Use XtDeviceDestroy to clean up the device. It is NOT an error when no default device is present.

This function may only be called from the main thread.
